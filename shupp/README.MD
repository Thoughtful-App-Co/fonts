## Shupp

A handwritten, casual typeface with a hand scribbled, w/ a thick sharpie pen type feel
![Shupp Specimen](./specimens/shupp_specimen.png)

### Details

| Property | Value |
|----------|-------|
| Style | Handwritten |
| Weight | Regular (400) |
| Glyphs | 94 |
| Format | OTF |

### Character Set

```
ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz
0123456789
!@#$%^&*(){}[]|:;<>?,./
```

### Installation

**Desktop:** Download `Shupp.otf` and install via your OS font manager.

**Web:**
```css
@font-face {
  font-family: 'Shupp';
  src: url('./fonts/Shupp.woff2') format('woff2'),
       url('./fonts/Shupp.otf') format('opentype');
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}
```

### Use Cases

- Blog headers
- Personal branding
- Signatures
- Casual UI elements
- Handwritten Journals and Blogs
